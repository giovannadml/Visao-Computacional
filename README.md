  ## Descrição do Projeto

  Utilização do OpenCV e da linguagem python para segmentar imagens, aplicar filtros por meio de convolução, adição de elementos (retas e textos), cálculo de contornos e áreas entre outros.

  ## Como usar

Para visualizar o arquivo .pyntb:


  - [ ] caso não esteja instalado ainda, realizar o download e a instalação do <a href="https://www.python.org/downloads/">python</a> no site oficial (selecionar a opção de adicionar o Python ao PATH);
  - [ ] no windows, entrar no terminal de comando (menu iniciar -> digite "cmd") e executar os comandos:

    - `pip install matplotlib`
    - `pip install opencv-python`
    - `pip install notebook`


  - [ ] no linux executar os comando pip no terminal usando pip3;
  - [ ] para abrir o arquivo .pyntb é necessário utilizar ou o Jupyter Notebook ou o Google Colab, neste sendo apenas nessesário acessar o link, entrar com a sua conta e abrir o arquivo NAC5.pyntb (Arquivo -> Abrir notebook -> Upload -> Escolher arquivo);
  - [ ] para instalar o Jupyter Notebook, abra o cmd e execute o comando `python -m pip install jupyter`;
  - [ ] depois de ter tudo instalado, abra um novo cmd e execute `jupyter notebook` depois de navegar até o diretório do projeto;
  - [ ] com o jupyter sendo executado, abra o arquivo .pyntb e execute as células uma por vez (selecionando a célula desejada e clicando em "Run") ou todas de uma vez (Kernel -> Restart & Run All -> Restart & Run All Cells).



Para o arquivo .py:


  - [ ] através do terminal de comando, navegar até o diretório onde o arquivo está localizado e apenas executar o comando `python webcam.py` (caso por algum motivo o nome do arquivo tenha sido modificado, será necessário trocar "webcam" pelo novo nome);
  - [ ] ;a câmera será ligada, assim é só ter a imagem de teste impressa para ver os contornos, a reta e o ângulo da reta em relação ao plano horizontal.
